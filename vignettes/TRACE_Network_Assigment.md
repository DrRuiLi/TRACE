After CN pattern detection and dynamic filtering, CN-hit features (seeds) were linked by MS-derived edges—CN labelling, adduct, isotope, and fragment relationships—into a candidate graph restricted to seed–seed pairs. Edges inconsistent with assigned CN formulas were removed (e.g. adduct edges require no change in CN label; fragment and isotope edges require matching C/N counts in the edge formula difference and the CN label change between endpoints). The graph was then partitioned into connected components by edge connectivity alone.

Within each component, global assignment evaluated all simple paths between node pairs (maximum length six by default). For a directed path $P = (v_1,\ldots,v_L)$, edge formula differences were oriented and summed to a net change $\Delta\mathrm{form}(P)$ (isotope labels collapsed to the light-element basis). A path was blocked (connection score zero) if $\Delta\mathrm{form}(P)$ was non-empty and matched an entry in the MS mass-difference lookup (theoretical adduct, fragment, and isotope differences). Otherwise the path score was $s(P) = \max\bigl(0,\, 1 - \sum_e |\varepsilon^{\mathrm{ppm}}_e|/\delta_{\mathrm{ppm}} - \sum_e |\Delta t_e|/\delta_{\mathrm{rt}}\bigr)$, where $\varepsilon^{\mathrm{ppm}}_e$ and $\Delta t_e$ are per-edge *m/z* error (ppm) and RT shift (s), and $\delta_{\mathrm{ppm}}$, $\delta_{\mathrm{rt}}$ are dynamic tolerances from the CN-hit versus background error distributions ($\hat{\sigma}\cdot\Phi^{-1}(0.999)$ and $\hat{\sigma}\cdot\Phi^{-1}(0.99999)$, respectively).

The pairwise connection score $S(u,v)$ was the maximum $s(P)$ over non-blocked simple paths from $u$ to $v$ ($S(u,u)=1$). Nodes were clustered by building an undirected compatibility graph with an edge between $u$ and $v$ when $S(u,v) \ge \theta$ (default $\theta = 0.5$) and the pair was not blocked; connected components defined assignment groups. Each feature received a group label and a representative seed (minimum feature ID in the group). Group labels were prefixed by connectivity-component index so that assignment groups were unique across the full seed network. These assignments were passed to downstream annotation to assign metabolite versus adduct roles and shared seeds within each compatible subnetwork (`TRACE_network_assignment` in the TRACE package).
